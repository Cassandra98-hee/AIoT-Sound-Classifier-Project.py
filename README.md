# IoT Sound Classifier
ESP32-based sound classification system with Blynk integration.

## Components
- ESP32 board, 1 microphone, 1 piezoelectric buzzer, 1 16x2 LCD (I2C), 2 buttons, 2LEDs, 1  1kÎ© resistor

## Features
- AI keyword extraction and classification
- Blynk remote monitoring
- Real-time sound visualization

## Simulation
Built and tested on [Wokwi](https://wokwi.com/)

## Files
- `sketch.ino` - Main code
- `model.h` - AI model
- `diagram.json` - Circuit diagram
